# 🎓 Öğrenci & Öğretmen Etkileşim Portalı

Bu web projesi, eğitim kurumlarında öğretmenlerin materyal paylaşımı yapabildiği, öğrencilerin ise ders takibi ve ödev kontrolü yapabildiği dinamik bir platformdur.

## 🚀 Temel Fonksiyonlar
* **Kullanıcı Yetkilendirme:** Öğretmen ve öğrenci rolleri için farklı panel erişimleri.
* **Ders Yönetimi:** Öğretmenlerin ders notu ve duyuru paylaşma özelliği.
* **Veritabanı Entegrasyonu:** Kullanıcı verileri ve ders içeriklerinin güvenli şekilde depolanması.
* **Responsive Tasarım:** Mobil ve masaüstü cihazlarla tam uyumlu arayüz.

## 🛠️ Kullanılan Teknolojiler
* **Frontend:** HTML5, CSS3, JavaScript (Bootstrap/Tailwind)
* **Backend:** [Buraya kullandığın dili yaz örn: ASP.NET Core veya PHP]
* **Database:** SQL Server / MySQL
